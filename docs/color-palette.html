Применение в UI

  • Бирюза — шапка, кнопки «Сохранить», вкладки, ссылки, импорт, подсказки типов
  • Красный — «+ Новый товар», активный пункт меню (полоска), вкладка карточки, предупреждения, ошибки
  • Ярко-зелёный — успешные сообщения, маркер «Активен», галочка «Товар активен»
  • Серый — границы полей и таблиц
